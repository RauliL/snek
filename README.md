# snek

Experimental programming language where Python meets TypeScript, kinda.

## How to compile?

You need to have [CMake] and [cget] installed on your system, then do this:

```bash
$ cget install
$ mkdir build
$ cd build
$ cmake ..
$ make
```

And tada!

[CMake]: https://cmake.org
[cget]: https://cget.readthedocs.io/en/latest/
